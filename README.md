# Student-Attendance-Tracker 
This is a Web application to help student to track their attendance. The site is accsesable from [here](https://pardhu-b-chowdary.github.io/Student-Attendance-Tracker/).  


Student can add their Subjects and Schedule. which is displayed as both in timetable format and Statistics for the each subject along with overalls.


This app is purely built on basic HTML, CSS and JavaScript. A port to run using ReactJs is in mist of building. This app use browsers local storage to save data making it so you can acces the data even if you close the tab and browser.  


Though browsers clear cache may delete the data, to counter that we can export the data into JSON from time to time and Reimport the JSON
